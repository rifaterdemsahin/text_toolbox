URL encoding (also called percent encoding) converts special characters into a format that can be safely used in URLs. For example:

  • Spaces become %20
  • & becomes %26
  • = becomes %3D
  • @ becomes %40

Common uses:

  • Creating query parameters in URLs
  • Passing special characters safely in web addresses
  • Building API requests
  • Sharing URLs with spaces or symbols

All processing happens in your browser. No data is sent to any server.
